# TWOTTER

## Description

A simple Twitter clone using [Vite](https://vitejs.dev) + [Tailwind CSS](https://tailwindcss.com) + [Firebase](https://firebase.google.com).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
